# Screenshots

Visual overview of Periodix-Planner's features and interface.

## Time Budgeting

![Time budgeting screenshot](/img/features/time-budgeting.png)

Allocate and track time across categories with hierarchical tracking from yearly down to daily periods. [Learn more →](/features/time-budgeting)

## Allocation Editor

![Allocation editor screenshot](/img/features/edit-allocator.png)

Interactive modal for editing time allocations with drag-to-adjust bars, quick-fill buttons, and keyboard shortcuts. [Learn more →](/features/allocation-editor)

## Time Budget Blocks

![Time budget blocks screenshot](/img/features/time-budget-blocks.png)

Embed time budget visualizations directly in your notes using code blocks. [Learn more →](/features/time-budget-blocks)

## Visual Statistics

![Visual statistics screenshot](/img/features/visual-statistics.png)

Get visual insights into your time allocation with interactive pie charts and detailed tables. [Learn more →](/features/visual-statistics)

## Note Generation

![Note generation screenshot](/img/features/note-generation.png)

Automatic periodic note generation that stays one step ahead by creating the next period's note in advance. [Learn more →](/features/note-generation)

## Note Interconnection

![Note interconnection screenshot](/img/features/note-interconnection.png)

All periodic notes are intelligently linked through frontmatter properties, creating a navigable web of time. [Learn more →](/features/note-interconnection)

## Navigation

![Navigation screenshot](/img/features/navigation.png)

Quick commands to navigate between periods with keyboard shortcuts and command palette integration. [Learn more →](/features/navigation)

## Categories

![Categories screenshot](/img/features/categories.png)

Organize your time investments with custom categories and color coding for visual identification. [Learn more →](/features/categories)

## Period Children

![Period children screenshot](/img/features/period-children.png)

View all child periods for a given period - see all days in a week, weeks in a month, months in a quarter, and quarters in a year. [Learn more →](/features/period-children)

## ActivityWatch Integration

![ActivityWatch integration screenshot](/img/features/activity-watch.png)

Automatically track and visualize your computer usage in daily notes with ActivityWatch integration. [Learn more →](/integrations)
