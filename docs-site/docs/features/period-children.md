# Period Children View

The Period Children view shows all child periods for a given period, helping you navigate and understand the hierarchical structure of your periodic notes.

## ðŸŽ¯ Overview

Period Children view displays:
- All days in a week
- All weeks in a month
- All months in a quarter
- All quarters in a year

## ðŸš€ Opening the View

### From Commands

1. Open a periodic note (weekly, monthly, quarterly, or yearly)
2. Use command: **"Show child periods"**
3. Modal opens showing all child periods

### Requirements

- Must be in a periodic note
- Period must have children (daily notes don't have children)
