- **Interactive** - Click to enlarge for detailed view
- **Real-time updates** - Reflects current allocations

### Enlarged View

Click the pie chart to open an enlarged modal view with:
- Larger, more detailed visualization
- Better visibility of all categories
- Full screen viewing option

## ðŸ“‹ Allocation Table

The allocation table provides detailed information:

### Columns

- **Category** - Name and color indicator
- **Allocated Hours** - Total hours allocated
- **Percentage** - Percentage of total available hours
- **Parent Budget** - Status relative to parent period budget
- **Child Allocated** - Hours allocated in child periods (if applicable)

### Sorting

Click column headers to sort by:
- Category name (alphabetical)
- Allocated hours (numerical)
- Parent budget status
- Child allocations
